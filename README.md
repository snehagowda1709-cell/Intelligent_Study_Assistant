
# Intelligent Study Assistant

## Overview
A Streamlit-based chatbot that uses your Gemini API key to answer study questions from uploaded PDF notes.

## Setup Instructions
1. Clone the repository or extract the ZIP folder.
2. Install dependencies:
```
pip install -r requirements.txt
```
3. Run the app:
```
streamlit run app.py
```
4. Upload your study PDFs and start asking questions.
